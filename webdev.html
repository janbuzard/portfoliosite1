  • Event Handlers
  • This used event handlers for all JavaScript events. The events are for toggling on and off the Hide Low Priority, Highlight Current Row and Hide Type Column buttons.
  • Adding Focus
  • Event handlers were used to changed the appearance of the form fields and submit button. All text added to the form is displayed in the preview area.
  • Event Delegation
  • This uses event listeners and event delegation. Mousing over the question mark gif toggles displaying help instructions. SHIFT-Click toggles locking and unlocking input on table cells. ALT-Click toggles a bolder border and toggles highlighting text in a table cell. CTRL-Click removes the value in the table cell. Any combination 2 or more of CTRL-Click, ALT-Click and SHIFT-Click makes the combined effects work.
  • Adding Functionality
  • This is the first project reprogrammed to use event delegation and event listeners instead of event handlers. In addition to the original events, validity of the URL is checked, instructions are added above each form, an add another button is displayed and when Add Anotion is clicked, an new input box is added until there are 5.
